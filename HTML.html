üîπ Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It describes the structure of web documents using tags and elements.

üîπ Basic Structure of an HTML Document

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>

üîπ Headings and Paragraphs

HTML provides six levels of headings, from <h1> to <h6>. Paragraphs are defined with the <p> tag.

üîπ Lists in HTML

  • Ordered List (<ol>)
  • Unordered List (<ul>)
  • Definition List (<dl>)

üîπ Links and Images

<a href="https://www.example.com">Visit Example</a>
<img src="image.jpg" alt="Sample Image">
