Introduction to C++ Programming

C++ is a powerful general-purpose programming language. It can be used to develop operating systems, browsers, games, and much more. C++ supports both procedural and object-oriented programming making it very flexible.

Structure of a C++ Program

Every C++ program has at least one function, which is main(). The program execution starts from main(). Example:

#include <iostream>
using namespace std;

int main() {
  cout << "Hello World";
  return 0;
}

Here #include <iostream> is a header file, using namespace std; allows usage of standard C++ objects, and cout is used for output.

Variables and Data Types

int age = 20;
float pi = 3.14;
char grade = 'A';
string name = "John";
bool isStudent = true;

Variables are storage locations identified by names. Data types define the kind of data variables can hold.

Operators in C++

C++ provides various operators:

  • Arithmetic Operators (+, -, *, /, %)
  • Relational Operators (<, >, ==, !=, <=, >=)
  • Logical Operators (&&, ||, !)
  • Assignment Operators (=, +=, -=, etc.)

These operators are used to perform computations and make decisions in programs.

Simple If Statement

int num = 10;
if (num > 5) {
  cout << "Number is greater than 5";
}

The if statement executes a block of code when the condition is true.

If-Else Statement

int num = 4;
if (num % 2 == 0) {
  cout << "Even number";
} else {
  cout << "Odd number";
}

If-else provides an alternative block when the condition is false.

If-Else-If Ladder

int marks = 85;
if (marks >= 90) {
  cout << "Grade A";
} else if (marks >= 75) {
  cout << "Grade B";
} else if (marks >= 50) {
  cout << "Grade C";
} else {
  cout << "Fail";
}

Used to check multiple conditions sequentially.

Nested If Statement

int a = 10, b = 20;
if (a != b) {
  if (a < b) {
    cout << "a is smaller";
  }
}

An if inside another if is called a nested if.

Switch Statement

int day = 3;
switch (day) {
  case 1:
    cout << "Monday";
    break;
  case 2:
    cout << "Tuesday";
    break;
  case 3:
    cout << "Wednesday";
    break;
  default:
    cout << "Invalid day";
}

The switch statement executes one block from multiple choices based on the value.

Loops

For Loop

for (int i = 1; i <= 5; i++) {
  cout << i << " ";
}

For loop is used when the number of iterations is known.

While Loop

int i = 1;
while (i <= 5) {
  cout << i << " ";
  i++;
}

While loop runs until the condition is false.
