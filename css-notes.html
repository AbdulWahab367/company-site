Complete CSS Reference’s

Quick notes for CSS selectors, keywords, properties, units and media queries

1. CSS Selectors

SelectorDescription
*Universal selector (all elements)
elementType selector (e.g. div, p)
.classClass selector
#idID selector
element elementDescendant selector
element > elementChild selector
element + elementAdjacent sibling selector
element ~ elementGeneral sibling selector
[attribute]Attribute selector
:hoverPseudo-class for hover
::before / ::afterPseudo-elements for adding content

2. CSS Keywords

KeywordUse
inheritInherit value from parent element
initialDefault value as defined by browser
unsetResets property to inherited or default
autoLet the browser calculate value
noneDisable property (e.g. display, border)
block / inlineDisplay type
flex / gridDisplay type
relative / absolute / fixed / stickyPosition types
transparentColor keyword

3. Common CSS Properties

PropertyDescription
colorText color
background-color / background-imageBackground styling
width / heightSize of element
padding / marginSpacing around element
borderBorder of element
font-size / font-family / font-weightFont styling
text-align / vertical-alignText alignment
line-heightLine spacing
displayElement display type
position / top / leftPositioning
z-indexStack order
overflowContent overflow control
visibility / opacityElement visibility
box-shadow / text-shadowShadow effects
transform / animation / transitionTransform & animations

4. CSS Value Units

UnitDescription
pxPixels (absolute unit)
%Percentage (relative unit)
emRelative to parent font size
remRelative to root font size
vh / vwViewport height/width
frFractional unit (grid)

5. Media Queries (Responsive CSS)

SyntaxDescription
@media screen and (max-width: 600px)Applies CSS on small screens
@media printApplies when printing
@media only screen and (min-width: 768px)Tablet or desktop view
