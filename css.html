üîπ Introduction to CSS

CSS (Cascading Style Sheets) ek style language hai jo HTML elements ko design aur layout dene ke liye use hoti hai. Ye colors, fonts, spacing, aur positioning control karti hai.

h1 {
  color: blue;
  font-size: 24px;
}

üîπ Types of CSS

  • Inline CSS (direct element ke andar)
  • Internal CSS (head tag ke style section me)
  • External CSS (alag .css file ke through)

üîπ CSS Selectors

  • Universal Selector (*)
  • Element Selector (p, h1, div)
  • Class Selector (.classname)
  • ID Selector (#idname)
  • Group Selector (h1, p)

üîπ Colors and Text Styling

  • Color names (red, blue, green)
  • Hex codes (#ff0000)
  • RGB/RGBA values
p {
  color: red;
  font-size: 18px;
  font-weight: bold;
}

üîπ CSS Box Model

Box model ke 4 parts hote hain:

  • Content
  • Padding
  • Border
  • Margin

üîπ Backgrounds and Borders

div {
  background-color: lightblue;
  border: 2px solid black;
}

üîπ Positioning and Layout

  • Static (default)
  • Relative
  • Absolute
  • Fixed
  • Sticky

üîπ Display and Visibility

  • block
  • inline
  • inline-block
  • none (hidden)

üîπ CSS Flexbox

.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
