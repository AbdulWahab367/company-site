Complete HTML Reference

A quick reference for HTML elements including structure, formatting, media, forms, semantics and scripting tags.

1. Basic Structure Tags

TagDescription
<!DOCTYPE>Defines the document type
<html>Root element of HTML document
<head>Contains metadata/information
<title>Specifies the document title
<body>Contains visible page content

2. Text Formatting Tags

TagDescription
<h1> - <h6>Headings
<p>Paragraph
<br>Line break
<strong>Bold (strong importance)
<em>Italic (emphasis)
<mark>Highlight text
<sub>Subscript
<sup>Superscript

3. Links, Media & Embeds

TagDescription
<a>Anchor (hyperlink)
<img>Image
<video>Video
<audio>Audio
<iframe>Embed another page
<object>Embed external object

4. List Tags

TagDescription
<ul>Unordered list
<ol>Ordered list
<li>List item
<dl>Definition list
<dt>Definition term
<dd>Term description

5. Table Tags

TagDescription
<table>Table container
<tr>Table row
<th>Table header
<td>Table cell
<thead>Head section
<tfoot>Footer section

6. Forms & Input

TagDescription
<form>Form container
<input>Input field
<textarea>Multi-line input
<button>Button
<select>Dropdown menu
<label>Form label
<fieldset>Group inputs

7. Semantic HTML5 Tags

TagDescription
<header>Top of page/section
<footer>Bottom of page/section
<nav>Navigation
<main>Main content
<section>Section content
<article>Independent content
<aside>Sidebar

8. Scripting & Metadata

TagDescription
<script>JavaScript code
<noscript>Fallback if JS disabled
<link>External CSS
<style>Internal CSS
<meta>Metadata

9. Other Tags

TagDescription
<div>Block-level container
<span>Inline container
<canvas>Draw graphics
<svg>Vector graphics
