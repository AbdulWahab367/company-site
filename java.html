üîπ Introduction to Java

Java is a popular, class-based, object-oriented programming language designed to have as few implementation dependencies as possible. It is widely used for building enterprise-scale applications, mobile apps, and more.

  • Platform Independent
  • Object-Oriented
  • Secure and Robust
  • High Performance

üîπ Setting Up Java Environment

To start programming in Java:

  • Install JDK (Java Development Kit).
  • Set up environment variables.
  • Use an IDE like IntelliJ IDEA, Eclipse, or VS Code.

üîπ Your First Java Program

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}

üîπ Java Variables and Data Types

int number = 10;
String name = "John";
boolean isJavaFun = true;

Java provides different types of variables, for example:

  • int - stores integers
  • double - stores floating-point numbers
  • String - stores text
  • boolean - stores true/false

üîπ Operators in Java

int x = 5 + 10; // Addition
int y = 20 - 5; // Subtraction
int z = 4 * 3;  // Multiplication
int w = 20 / 4; // Division

üîπ Conditional Statements

int age = 18;
if (age >= 18) {
  System.out.println("You are an adult.");
} else {
  System.out.println("You are not an adult.");
}

üîπ Switch Statement

int day = 3;
switch (day) {
  case 1:
    System.out.println("Monday");
    break;
  case 2:
    System.out.println("Tuesday");
    break;
  case 3:
    System.out.println("Wednesday");
    break;
  default:
    System.out.println("Other day");
}

üîπ Loops in Java

for (int i = 1; i <= 5; i++) {
  System.out.println(i);
}
int i = 1;
while (i <= 5) {
  System.out.println(i);
  i++;
}

üîπ Functions (Methods)

public class MyClass {
  static void myMethod() {
    System.out.println("Hello from method!");
  }

  public static void main(String[] args) {
    myMethod();
  }
}
