Introduction to JavaScript

JavaScript is a powerful and widely-used programming language that enables dynamic content, interactive features, and rich web experiences. It is essential for front-end development and widely used in back-end as well.

Writing JavaScript

// Inline JavaScript
<script>
  console.log("Hello World");
</script>
      
// Internal Script
<script>
  let a = 5;
  console.log(a);
</script>
      
// External File
<script src="script.js"></script>

JavaScript can be written inline, internally inside the HTML file, or in a separate external file. It is recommended to keep JS in a separate .js file for better code management.

Variables and Data Types

Variables are containers for storing data values. In JavaScript, we declare variables using var, let, or const.

let age = 25;
var name = "John";
const pi = 3.14;

Use let for variables that may change, const for constants, and avoid using var in modern JS.

Operators in JavaScript

Operators are symbols used to perform operations on values and variables.

// Arithmetic Operators
let sum = 5 + 10;
let diff = 20 - 5;
let mul = 4 * 3;
let div = 10 / 2;

// Comparison Operators
console.log(5 == "5");  // true
console.log(5 === "5"); // false

// Logical Operators
let x = true && false; // false
let y = true || false; // true
let z = !true;         // false

Control Flow: if Statement

let num = 10;
if (num > 5) {
  console.log("Number is greater than 5");
} else {
  console.log("Number is 5 or less");
}

The if statement executes a block of code if a condition is true. Use else for alternative actions and else if for multiple conditions.
