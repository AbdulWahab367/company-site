üìò What is SQL?

SQL (Structured Query Language) ek programming language hai jo databases ko manage karne ke liye use hoti hai. Ye data ko insert, update, delete, aur retrieve karne ke liye commands provide karti hai.

  • SQL = Structured Query Language
  • Database ke saath interact karne ke liye use hoti hai
  • Insert, Update, Delete, aur Retrieve kar sakte hain
  • Almost saare DBMS jaise MySQL, Oracle, PostgreSQL me use hoti hai

Basic SQL Commands

Yeh sabse common SQL commands hain:

  • CREATE ‚Äì Database aur Tables banane ke liye
  • INSERT ‚Äì Data add karne ke liye
  • SELECT ‚Äì Data retrieve karne ke liye
  • UPDATE ‚Äì Data update karne ke liye
  • DELETE ‚Äì Records delete karne ke liye

Creating Tables and Databases

CREATE DATABASE mydb;

CREATE TABLE users (
  id INT PRIMARY KEY,
  name VARCHAR(100),
  email VARCHAR(100)
);

Is tarah hum apna database aur tables bana sakte hain.

Data Types in SQL

  • INT ‚Äì Number
  • VARCHAR ‚Äì Text
  • DATE ‚Äì Dates
  • BOOLEAN ‚Äì True/False
CREATE TABLE products (
  id INT,
  name VARCHAR(50),
  price DECIMAL(10,2)
);

Retrieving Data with SELECT

SELECT * FROM users;

SELECT name, email FROM users;

SELECT * FROM users WHERE id = 1;

Filtering with AND/OR

SELECT * FROM users 
WHERE age > 18 AND city = 'Delhi';

Sorting and Limiting Data

SELECT * FROM users ORDER BY name ASC;

SELECT * FROM users LIMIT 5;

Inserting Data

INSERT INTO users (id, name, email) 
VALUES (1, 'Ali', 'ali@gmail.com');

Updating Records

UPDATE users 
SET name = 'Ahmed' 
WHERE id = 1;

Deleting Records

DELETE FROM users WHERE id = 1;
